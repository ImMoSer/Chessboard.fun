# 🧩 Спецификация: Адаптивная система тренировочного плана (V1.1)

Этот документ является **единым источником истины** для логики backend и отображения на frontend.

---

# 1. 🎯 Стрик-механика и прогресс

## Определение «успеха» (Auto-Complete)

_Истинный успех_ — это полное выполнение всех задач, определённых в `tasks_json` за день.

- **Триггер:** При каждом вызове `GET /api/training-plan/current` автоматически проверяется прогресс
- **Проверка:** Общее количество решённых задач (`puzzles_solved`) в `daily_mode_stats` должно быть **не меньше целевого значения (`count`) для каждого режима, подрежима и темы**
- **Автоматизация:** При достижении 100%:
  - устанавливается `is_completed: true`
  - увеличивается `current_streak`
  - обновляется `last_completed_plan_date`

➡️ Никаких ручных действий не требуется.

---

## Фазы стрика и стратегия

Выбор тем следует циклу в 30 дней:

| Фаза          | Дни   | Стратегия                                    |
| :------------ | :---- | :------------------------------------------- |
| **Discovery** | 0–9   | Исследование всего пула тем                  |
| **Weakness**  | 10–19 | Фокус на темах с наименьшим процентом успеха |
| **Strength**  | 20–29 | Фокус на сильных темах (закрепление)         |

---

## Grace Period (логика догонки)

- План за **вчера остаётся активным**
- Если задачи за вчера выполнены сегодня:
  - они полностью засчитываются
  - стрик сохраняется

- После завершения «догоняющего» плана:
  - можно сразу сгенерировать план на текущий день

# 2. 📊 Логика объёма (задачи в день)

## 🧠 Основной принцип

- **Количество задач на тему остаётся постоянным**
- **Сложность растёт только за счёт увеличения количества тем**

Формула:

`Общий объём = (задачи на тему) × (количество тем) × (количество подрежимов)`

# 4. 🚀 Прогрессия и апгрейд уровней

- **Повышение уровня:**
  - Novice → Pro → Master после **30 полностью завершённых дней**

- **Сброс:**
  - Стрик сбрасывается на **0** при переходе на новый уровень

- **Бесконечный режим:**
  - На уровне **Master** дальнейшего сброса после 30 дней нет

---

# ✅ Итог

- Прогресс основан на **полном выполнении дневного плана**
- Сложность растёт за счёт **увеличения количества тем, а не задач**
- Система адаптируется динамически через:
  - стрик
  - процент успешности
  - фазы тренировки
